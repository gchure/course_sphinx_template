Lesson 1: Example Notebook With Code

This is an example lesson generated via a Jupyter notebook that has lines of code executed within the notebook. In this case, we generate new a new cell of Python code where we import some standard stuff.

[1]:

# Import modules needed to execute the code blocks
import numpy as np
import seaborn  as sns
sns.set()

# Make a plot
samples = np.random.normal(size=1000)
sns.histplot(samples)
